이미지를 표시하는 태그(Image 태그)
	* <img />태그
	* 컨텐츠를 가지지 않는 태그

	* 태그의 주요 속성
	   <img src="" alt="" width="" height="" />
		- src    : 표시할 이미지 파일의 경로와 파일명을 정의한다.
		- alt    : 이미지에 대한 설명을 정의한다.(웹접근성)
		- width  : 이미지의 너비를 정의한다.(px단위)
		- height : 이미지의 높이를 정의한다.(px단위)

- 주요태그들 -
html 문서 : html, head, body
제목 : h1 ~ h6
본문의 내용 : p
포맷팅된 텍스트 : strong, em, mark, del, small
목록 : ul>li, ol>li, dl>dt/dd
테이블 : table, thead, tbody, tr, th, td
링크 : a <--- href, target
이미지 : img <--- src, alt, width, height
