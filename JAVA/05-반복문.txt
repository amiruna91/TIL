1.for문
   지정된 횟수만큼 블록내의 수행문을 실행시킨다.
  반복횟수의 지정은 초기식, 조건식, 증감식을 사용해서 조절한다.
  for (초기식; 조건식; 증감식) {
 	수행문;
  }
  * 초기식 : 반복의 횟수를 지정하기 위한 변수를 초기화 한다.
  * 조건식 : 조건식의 연산결과가 true로 판정되는 동안 반복을 지속한다.
  * 증감식 : 반복문을 제어하기 위해서 초기식에서 선언한 변수의 값을 변경시킨다.

 예시:
  for (int i = 1; i <= 3; i++) {
	sysout("안녕");
  }
  
	반복횟수	초기식		조건식			수행문	 수행문 실행중 i값	증감식
 	1		i = 1		i<=3	-> true		안녕		1		i = 2
  	2				i<=3	-> true		안녕		2		i = 3
  	3				i<=3	-> true  	안녕		3		i = 4
  	4				i<=3	-> false  ->    반복중단
2.break 와 continue
	break :
		반복문을 탈출한다. for문, while문 안에서 사용된다.

	continue :
 		반복문안에 있는 수행문을 실행하다가 continue가 실행되면 남아있는 수행문의 실행을 취소하고 
	       증감식으로 이동해서 다음 번 반복을 하러 간다.
