CSS의 선택자
	- 기본 선택자
	---------------------------------------------------------------------------------------------------------
	선택자		작성예			설명
	---------------------------------------------------------------------------------------------------------
	*		* {style}		모든 엘리먼트를 선택.
	tag		p {style}		모든 p 엘리먼트를 선택.
	.class		.hot {style}		class속성값에 hot을 포함하고 있는 모든 엘리먼트를 선택.
	#id		#wrapper {style}	id="wrapper"인 엘리먼트 하나 선택.
	tag, tag	div, p {style}		모든 div, 모든 p 엘리먼트를 선택.
	.class, .class	.hot, .new {style}	class속성값에 hot 혹은 new를 포함하고 있는 모든 엘리먼트를 선택.
	tag.class	div.hot {style}		div중에서 class속성값이 hot을 포함하고 있는 모든 엘리먼트를 선택.
	.class.class	.hot.new {style}	class속성값에 hot과 new를 모두 포함하고 있는 모든 엘리먼트를 선택.
	----------------------------------------------------------------------------------------------------------
	- 조합 선택자 (공백, >, +, ~ 등의 기호를 이용해서 2개 이상의 기본선택자를 조합시킨 것)
	---------------------------------------------------------------------------------------------------------
	선택자		작성예			설명
	---------------------------------------------------------------------------------------------------------
	공백(후손 선택자)
	tag tag		div p			div의 후손 중에서 모든 p를 선택.
	.class .class	.news .hot		class에 news를 포함하고 있는 엘리먼트의 후손 중에서 class에 hot을 포함하고 있는 것 전부 선택.					
	.class tag	.news p			class에 news를 포함하고 있는 엘리먼트의 후손 중에서 모든 p를 선택.
	#id tag		#footer p		id="footer"인 엘리먼트의 후손 중에서 모든 p 선택.
	#id .class 	#header .menu		id="header"인 엘리먼트의 후손 중에서 class에 menu를 포함하고 있는 것 전부 선택.

	> (자식 선택자) * 5가지의 조합중 하나만 예시로 작성함.
	tag > tag	div > p			div의 자식 중에서 모든 p를 선택
	
	+ (인접한 형제 선택자)
	tag + tag	h3 + p			h3 바로 다음에 있는 p를 선택
	
	~ (형제 선택자)
	tag ~ tag	h3 ~ p			h3 다음에 있는 모든 p를 선택
	---------------------------------------------------------------------------------------------------------
	
	- 속성 선택자
	---------------------------------------------------------------------------------------------------------
	선택자		작성예			설명
	---------------------------------------------------------------------------------------------------------
	[attr]		[name]			name속성을 가진 모든 엘리먼트를 선택.
	[attr="val"]	input[type="text"]	input태그 중에서 type="text"인 모든 엘리먼트 선택.
	[attr^="val"]	a[href^=http"]		a태그 중에서 href속성값이 http로 시작하는 모든 엘리먼트 선택.
	[attr$="val"]	a[href$="net"]		a태그 중에서 href속성값이 net으로 끝나는 모든 엘리먼트 선택.
	[attr*="val"]	a[title*="학교"]	a태그 중에서 title속성값에 "학교"라는 문자를 포함하고 있는 모든 엘리먼트 선택. 
	[attr~="val"]	a[title~="학교"]	a태그 중에서 title속성값에 "학교"라는 문자를 공백으로 구분된 단어로 포함호고 있는 모든 엘리먼트 선택.
	---------------------------------------------------------------------------------------------------------
	
	- 필터 선택자
	---------------------------------------------------------------------------------------------------------
	선택자		작성예			설명
	---------------------------------------------------------------------------------------------------------
	:fisrt-child	p:fisrt-child		p중에서 부모를 기준으로 첫번째 자식 엘리먼트인 p 선택.
	:last-child	p:last-child		p중에서 부모를 기준으로 마지막번째 자식 엘리먼트인 p 선택.
	:nth-child(n)	p:nth-child(2)		p중에서 부모를 기준으로 두번째 자식 엘리먼트인 p 선택.
			p:nth-child(odd)	p중에서 부노를 기준으로 홀수번째 자식 엘리먼트인 모든 p 선택.
			p:nth-child(even)	p중에서 부모를 기준으로 짝수번째 자식 엘리먼트인 모든 p 선택.
	:not(selector)	p:not(:fisrt-child)	p중에서 부모를 기준으로 첫번째 자식이 아닌 모든 p 선택.
	---------------------------------------------------------------------------------------------------------

	- 가상선택자
	---------------------------------------------------------------------------------------------------------
	선택자		작성예			설명
	---------------------------------------------------------------------------------------------------------
	:hover		a:hover			a중에서 마우스포인트가 올라가 있는 것 선택
	:link		a:link			a중에서 방문한 적이 없는 링크
	:visited	a:visited		a중에서 방문한 적이 있는 링크
	:active		a:active		a중에서 지금 클릭하는 링크
	
	:hover		div:hover		div중에서 마우스포인트가 올라가 있는 것 선택
	---------------------------------------------------------------------------------------------------------

	- 선택자의 우선순위
		- 선택자의 우선순위는 하나의 엘리먼트에 서로 다른 두개 이상의 선택자에 의해 동일한 CSS 선언이
		  적용되고 있을 때 어느 선택자가 우선순위가 높은지를 구별하는 것이다.
		- 선택자 우선순위 계산 방식
			* 1순위 : !important를 포함하고 있는 CSS 선언
			* 2순위 : inline CSS방식으로 지정한 CSS 선언
			* 3순위 : #id선택자로 지정한 CSS 선언
			* 4순위 : 클래스/속성/필터/가상 선택자로 지정한 CSS 선언
			* 5순위 : 태그 선택자로 지정한 CSS 선언 
			* 6순위 : 전체 선택자로 지정한 CSS 선언
