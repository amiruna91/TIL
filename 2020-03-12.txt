  과 반복문'이 있다.

1.if문 
  - 제시된 조건식의 연산결과에 따라서 특정 수행문의 실행여부를 결정한다.
  1) if (조건식) {
	  조건식의 연산결과가 true로 판정될 때 실행될 수행문;
     }
  
  -if ~ else문
  2) if (조건식) {
 	  조건식이 연산결과가 true로 판정될 때 실행될 수행문;
     } else {
      	  조건식의 연산결과가 false로 판정될 때 실행될 수행문;
     }

  3)if ~ else if ~ else if ~ else 문
    3개 이상의 경우의 수를 처리해야 할 때 사용되는 조건문이다.
     if (조건식1) {
     	조건식1이 true로 판정될 때 실행된다. 이후 문장은 실행되지 않는다.
     } else if (조건식2) {
  	조건식2가 true로 판정될 때 실행된다. 이후 문장은 실행되지 않는다.
     } else if (조건식3) {
  	조건식3이 true로 판정될 때 실행된다. 이후 문장은 실행되지 않는다.
     } else {
  	조건식1, 조건식2, 조건식3이 false로 판정되면 실행된다.
     }
   4)중첩 if문
     if문의 블록내에 다른 if문을 포함시키는 것\

2.switch문
  비교값과 일치하는 다양한 경우를 간결한 표현으로 처리할 수 있다.

 	switch(비교값) {
  		case 값1:
  			비교값이 값1과 일치하는 경우 실행되는 수행문;
			break;
  		case 값2:
  			비교값이 값2과 일치하는 경우 실행되는 수행문;
			break;
  		default:
  			비교값이 제시된 값과  일치하는 것이 없을 때 실행되는 수행문;
  	}
  	* 비교값으로 사용가능한 값은 정수, 문자, 문자열 만 가능하다.
  	* case에 제시된 값은 상수 혹은 리터럴값이어야 한다.
	* break문은 switch 문을 벗어나게 한다.

3.임의의 정수 만들기
  // 주사위는 1 ~ 6사이의 정수값이 랜덤하게 나와야 한다.
  // 최소값: 1 최대값: 6
  // (int) (Math.random() * 최대값 + 최소값)
  int ran = (int) (Math.random() * 6 + 1);
  System.out.println(ran);

4.for문
   지정된 횟수만큼 블록내의 수행문을 실행시킨다.
  반복횟수의 지정은 초기식, 조건식, 증감식을 사용해서 조절한다.
  for (초기식; 조건식; 증감식) {
 	수행문;
  }
  * 초기식 : 반복의 횟수를 지정하기 위한 변수를 초기화 한다.
  * 조건식 : 조건식의 연산결과가 true로 판정되는 동안 반복을 지속한다.
  * 증감식 : 반복문을 제어하기 위해서 초기식에서 선언한 변수의 값을 변경시킨다.

 예시:
  for (int i = 1; i <= 3; i++) {
	sysout("안녕");
  }
  
	반복횟수	초기식		조건식			수행문	 수행문 실행중 i값	증감식
 	1		i = 1		i<=3	-> true		안녕		1		i = 2
  	2				i<=3	-> true		안녕		2		i = 3
  	3				i<=3	-> true  	안녕		3		i = 4
  	4				i<=3	-> false  ->    반복중단
