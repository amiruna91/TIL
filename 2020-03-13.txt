1.break
 반복문을 탈출한다. for문, while문 안에서 사용된다.

2.continue
 반복문안에 있는 수행문을 실행하다가 continue가 실행되면 남아있는 수행문의 실행을 취소하고 
증감식으로 이동해서 다음 번 반복을 하러 간다.

3.중첩 for문
 for (int i = 1; i <= 3; i++) {
	System.out.println("i값: " + i);
			
	for (int j =1; j <= 5; j++) {
		if (j >= 3) {
			break; 	// 중첩for문에서 break는 해당 for문을 탈출한다.
				// 바깥쪽 for문은 영향을 받지 않는다.
 }
